# Code Of Conduct

This project welcomes practical, respectful collaboration from timing system
vendors, clubs, archivists, developers, competitors, and data users.

Participants are expected to:

- be respectful and constructive
- focus feedback on the format, examples, tools, and documentation
- assume good intent while still naming concrete problems clearly
- avoid harassment, insults, or personal attacks
- avoid posting private or sensitive personal data

Maintainers may edit, hide, or remove comments and contributions that are
abusive, off-topic, or unsafe for a public specification project.

If you see a conduct concern, report it through GitHub issues if appropriate or
through the official AutocrossRank.com contact path.
