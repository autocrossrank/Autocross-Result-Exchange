# Security

This repository is primarily a public data format specification.

Do not include private driver contact information, private identifiers, or
sensitive personal data in example files, issues, or pull requests.

Do not submit real private data unless it is already publicly available in
published results and appropriate to include in a public repository. When in
doubt, use fictionalized examples.

For security issues with AutocrossRank.com, contact through the official
AutocrossRank site or GitHub issues if appropriate.

For issues with this repository's validation tooling or workflows, open a GitHub
issue with enough detail to reproduce the problem.
